# План сдачи итогового проекта — курс Zerocoder по Claude Code

## Что сдаём

**Telegram-бот для анализа фриланс-заказов** — инструмент для карьерного центра Zerocoder.  
Бот принимает текст заказа и за несколько секунд делает всё, что раньше делалось вручную: определяет стек, проверяет соответствие скиллсету, пишет отклик, оформляет для базы или Telegram-канала с баннером.
