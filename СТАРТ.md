# Как запустить бота

## Шаг 1 — Получить ключ Anthropic API

1. Зайди на сайт: **console.anthropic.com**
2. Войди или зарегистрируйся
3. В левом меню выбери **"API Keys"**
4. Нажми **"Create Key"**, дай любое название
5. Скопируй ключ — он начинается с `sk-ant-...`
6. Открой файл `.env` в папке проекта и вставь ключ в строку:
   ```
   ANTHROPIC_API_KEY=sk-ant-...твой ключ...
   ```

---

## Шаг 2 — Запустить бота

Открой **cmd** или **PowerShell** и выполни по порядку:

```
cd C:\Users\el02s\Documents\analyz_agent
py -3.11 -m bot.main
```

Бот запустится и будет работать, пока открыто это окно.

---

## Шаг 3 — Пользоваться ботом

1. Найди своего бота в Telegram (по токену, который создавала через @BotFather)
2. Напиши `/start`
3. Отправь текст любого фриланс-заказа (или ссылку на него)
4. Выбери нужное действие из кнопок

---

## Что делать если бот упал

Просто снова запусти команду из Шага 2.
Если запущено несколько копий — закрой все окна терминала и запусти заново.
